# 🏰 Strategy Game (Windows)

A mini-game in the turn-based strategy genre, where the player controls a castle and competes with opponents.


## Features
- Control your castle and manage resources.
- Recruit and command units.
  
## 🛠️ Tech Stack
- **Languages:** C#
- **Frameworks:** .NET 4.7.2 (WinForms) 

## 🚀 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/OPivnieva/strategy-game.git
   ```
2. Open the project in your Visual Studio.
3. Build and run the game.
4. Or download the game in Release.
